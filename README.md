Shareable [TypeScript ESLint](https://github.com/typescript-eslint/typescript-eslint) config.

## Installation

```sh
npm i --save-dev eslint-config-binden-ts
```

## Usage

- Add the `extends` property (in the `.eslintrc.json` file)

```json
{ "extends": "binden-ts" }
```
